## Vegansk Scraper

Den veganska scrapern skrapar veganska hemsidor efter namn på recept med medföljande länk.
Recept och receptlänk läggs in i en databas.

#### package dependencies:
  "ejs": "^2.4.2",  
  "express": "^4.13.4",  
  "leveldown": "latest",  
  "levelup": "latest",  
  "x-ray":  "lastest"
